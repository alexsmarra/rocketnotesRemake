1- Abrir o terminal (fora do vscode) e na pasta desejada    npm create vite@latest   e depois
proceder como dito no passo a passo pelo próprio terminal

2- para executar o projeto   npm run dev

3- npm install --save styled-components   para instalar nosso processador / organizador de css,
para escrevermos css dentro de js (caso o vscode não reconheça, instalar extensão vscode styled
components em nosso vscode)
